
# Gutenberry

frontend design for a blog

used tools

- jQuery
- bootstrap
- bs-custom-file-input

    ```html
    <script src="dist/bs-custom-file-input.js"></script>
    ```

    Initialize the plugin on document ready.

    ```js
    $(document).ready(function () {
        bsCustomFileInput.init()
    })
    ```
- font `Montserrat-Regular`
